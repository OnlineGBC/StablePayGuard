# Contributing

Thank you for your interest in contributing to Codex PayRails Agent.

## Workflow

1. Fork the repository
2. Create a feature branch

```bash
git checkout -b feature/my-feature
```

3. Make your changes
4. Commit using a clear message

```bash
git commit -m "Add feature"
```

5. Push your branch

```bash
git push origin feature/my-feature
```

6. Open a pull request

## Guidelines

- Follow PEP 8 for Python code
- Keep HTML/CSS changes visually consistent with the dashboard theme
- Document new API endpoints in `README.md`
- Add tests where practical
- Keep commits focused and readable

## Pull Request Checklist

- code runs locally
- no secrets committed
- README updated if behavior changed
- new configuration documented
- formatting checked
